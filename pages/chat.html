Gitalk 设置指南

要启用评论功能，请按照以下步骤操作：

  1. 在 GitHub 上创建一个 OAuth App：
    • 访问 GitHub OAuth Apps
    • 填写应用名称（如 "Blog Comments"）
    • 主页 URL 填写您的博客地址（如 "https://zhangwenniu.github.io"）
    • 授权回调 URL 也填写您的博客地址
    • 点击 "Register application"
  2. 获取 Client ID 和 Client Secret
  3. 在 _config.yml 文件中更新 Gitalk 配置： 
    gitalk:
  enable: true
  clientID: '您的 Client ID'
  clientSecret: '您的 Client Secret'
  repo: 'zhangwenniu.github.io'
  owner: 'zhangwenniu'
  admin: ['zhangwenniu']

设置完成后，删除此指南即可。
