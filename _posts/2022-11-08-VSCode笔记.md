# 快捷键

## 检索函数、变量

```
Ctrl+Shift+O
```

## 搜索文件夹下的代码

```
Ctrl+Shift+F
```



## 查看变量的定义、声明、引用

```
Shift+F12
```

# 插件

## Outline Map

支持追踪光标、自动跟踪所在代码模块。建议拖动移动到右侧侧边栏。
